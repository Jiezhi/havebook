# HaveBook，姑且叫做『有书』吧

> 由来：因为自己在各个平台买的书很多，比如实体书加上在多看、当当、京东以及现在排版「还比较差」的微信读书上买的电子书。
>
> 书一多就比较混乱，所以索性尝试做个程序将数据整合起来。
